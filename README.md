# EduSphere – Student Blogging Platform

EduSphere is a full-stack blogging platform where students can create, publish, and interact with blog posts.

## Features
- User authentication (Login / Signup)
- Create and delete blog posts
- Like and save posts
- Student profile system
- Profile photo upload
- View saved posts

## Tech Stack

Frontend
- Next.js
- Tailwind CSS

Backend
- Django
- Django REST Framework

Database
- PostgreSQL

## Installation

### Backend
git clone https://github.com/ash1k-mohammed/edusphere-backend
cd edusphere-backend
pip install -r requirements.txt
python manage.py runserver

### Frontend
git clone https://github.com/ash1k-mohammed/edusphere-frontent
npm install
npm run dev
